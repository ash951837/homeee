NIGHTFALL STUDIO 6 — GITHUB PAGES KURULUMU

1. ZIP dosyasını bilgisayarında çıkart.
2. ZIP'in kendisini değil, içindeki bütün dosyaları GitHub deposunun ana dizinine yükle.
3. index.html ve oyun.html aynı klasörde olmalı.
4. GitHub > Settings > Pages bölümünde:
   Source: Deploy from a branch
   Branch: main
   Folder: /(root)
5. Kaydet ve sayfayı yenilerken Ctrl+F5 kullan.
