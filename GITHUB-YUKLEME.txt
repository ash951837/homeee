GITHUB YUKLEME

1. Bu klasörün içindeki tüm dosyaları repository ana dizinine yükle.
2. GitHub Pages kullanıyorsan Settings > Pages bölümünden branch seç.
3. PWA ana açılış sayfası index.html, oyun sayfası oyun.html dosyasıdır.
4. Eski cache takılırsa telefonda site ayarlarından site verilerini temizle veya yeni gizli sekmede dene.
