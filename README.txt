GITHUB PAGES YUKLEME

1. Bu klasorun icindeki index.html, .nojekyll ve assets klasorunu repository kokune yukleyin.
2. GitHub > Settings > Pages bolumune gidin.
3. Source: Deploy from a branch, Branch: main, Folder: /(root) secin ve Save basin.
4. Dosya adi mutlaka index.html olarak kalmali.

Not: assets klasorunun konumunu degistirmeyin.
