# GitHub Pages kurulumu

1. Bu ZIP dosyasını bilgisayarında çıkart. ZIP dosyasının kendisini repoya yükleme.
2. `index.html`, `oyun.html`, `404.html`, `.nojekyll` ve `oyun` klasörünü deponun ana dizinine yükle.
3. GitHub deposunda **Settings → Pages** bölümüne gir.
4. **Build and deployment → Source** alanını **Deploy from a branch** yap.
5. Branch olarak `main`, klasör olarak `/(root)` seç ve **Save** düğmesine bas.
6. Yayın adresi birkaç dakika içinde Pages bölümünde görünür.

Ana sayfa: `https://KULLANICI.github.io/DEPO/`
Oyun: `https://KULLANICI.github.io/DEPO/oyun/`

Önemli: Dosya adlarında büyük/küçük harf değişikliği yapma. `oyun` klasörünü eksiksiz yükle.
